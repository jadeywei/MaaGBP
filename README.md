<p>20260310记录<br/>
实现了：

##### 打开游戏、点击Tap To Start

##### 每日首次登录回主页

当前仅识别‘下一步’按钮<br/>

##### 消耗体力

识别体力十位数 若为1或者2 则进行演出<br/>
（待完善）仅识别‘自由演出’按钮，未做活动按钮识别<br/>
‘自动演出 关’点击 改为开<br/>
十倍体力，识别‘最大消费还’五个字<br/>
等待120秒auto完成。<br/>
完成后，’下一步‘和’确定‘同时找，同时点。<br/>
打歌完成后，点确定-菜单-跳过-屏幕中间跳过，回主页。<br/>

##### 领礼包

礼包按钮 右上角有红点则点进去-若显示’没有可领取的礼物‘则返回。否则点’键领取‘-确定-再点确定(-此时显示’没有可领取的礼物‘)<br/>


##### 领任务奖励

右上角有红点 循环：“点红点-点领取-点ok”。直到找不到红点，点左上角返回<br/>

##### 每日3次免费抽卡

若有’次免费‘，进入抽卡-左侧下滑3下-点免费-点招募-点SKIP两次-点中间确定<br/>
先判断是否有’免费‘（进入第二次抽卡），有则进入招募-点SKIP两次的流程。<br/>
否则点确定（3次抽完）-点左上角返回<br/>

##### 每周星石

菜单有红点-星石商店和红点-超值组合和红点-免费按钮-领取按钮-已领取按钮-左上角返回<br/>
